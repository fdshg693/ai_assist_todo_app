## システム説明

### AI Assist Todo App
- **目的**: Todoタスクの効率的な管理とAIによる支援
- **主な機能**:
  - Todo CRUD操作（作成・編集・削除・完了管理）
  - AIレコメンド（次に取り組むべきタスクの提案）
  - AIサマリ（タスク状況の要約と進捗分析）
  - 優先順位管理

### 外部システム

#### Azure OpenAI Service
- **役割**: AIモデルの提供
- **使用モデル**: gpt-5-nano（デフォルト）、gpt-5、gpt-4o-mini等
- **プロトコル**: Response API
- **用途**: 
  - タスクの優先順位付けとレコメンド
  - タスク状況のサマリ生成
  - 将来的なタスク分解や学習リソース生成

### ユーザー

#### エンドユーザー
- **役割**: アプリケーションの利用者
- **主な操作**:
  - Todoの作成・管理
  - AI推奨タスクの確認
  - タスクサマリの閲覧
