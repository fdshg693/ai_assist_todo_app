# docs フォルダの概要

- `docs/`: プロジェクトに関連するドキュメントを格納するフォルダ。以下のサブフォルダとファイルが含まれています。編集時は必ず元ファイルの粒度を保ち、1 ファイルに収めるのが適切でない場合（400 行を超える場合など）は、適切にファイルを分割してください。
  - `docs/app/CURRENT.md`: アプリケーションの現在の状態を記録するドキュメント。実装されている機能や進捗状況を簡潔にまとめる。
  - `docs/plan/`: プロジェクトの計画に関するドキュメントを格納するフォルダ。編集は行わないでください。
  - `docs/tech/`: プロジェクトで使用されている技術やツールに関するドキュメントを格納するフォルダ。新しい技術が導入された場合や既存の技術に変更があった場合に更新される。
