# 人間の開発者用ドキュメント

ここには、人間の開発者がプロジェクトに貢献するためのガイドラインや情報が含まれています。
AIコーダーに読み込ませる内容はここには含まれていません。

## [プロジェクト全体](./app/)

現状のプロジェクト全体の概要、構成、主要なコンポーネントについて説明しています。

## [アプリ開発プラン](./plan/)

アプリケーションの開発計画、タスクの詳細について記載しています。

## [技術スタック](./tech/)

現状のプロジェクトで利用されている技術スタックについて説明しています。
技術スタックの概要および、どのようにしてプロジェクトに適用されているかを理解するのに役立ちます。
